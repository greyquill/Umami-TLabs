Follow these simple steps to create a print-ready PDF from any page:

  1. Click on any page below to open it in a new tab
  2. Press Ctrl + P (Windows) or Cmd + P (Mac) to open the print dialog
  3. Select "Save as PDF" as the destination/printer
  4. Choose these settings:
    • Paper size: A4
    • Margins: None
    • Background graphics: Enabled
  5. Click Save to download the PDF

Tip: You can combine all individual PDFs using online tools like Adobe Acrobat or PDF Merge.
